Welcome to InsightVR CRM
To get started, connect your Supabase database. You only need to do this once — your credentials will be saved in this browser.
Where to find these:
1. Go to supabase.com and create a free project
2. In your project, go to Settings → API
3. Copy the Project URL and anon/public key below
4. Also run the SQL setup script (provided in the README)
